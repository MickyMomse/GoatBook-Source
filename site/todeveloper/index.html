    给开发者

    开发者您好
    现在目录结构大变样，请按照我写好的md文件进行编写
    不要使用老式的数字锚点，改为使用文字锚点如我 现在为./name/name.md#me
    编辑模式下花括号（大括号{}）中的内容就是锚点 以下是在各个目录引用name文件的方法

    目录 引用方案
    docs(主目录) ./name/name.md 或 ./name/name_thing.md
    name name.md 或 name_thing.md
    thing ../name/name.md 或 ../name/name_thing.md
